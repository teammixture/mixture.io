Important changes

We’re about to push an important update to Mixture (Windows v242 and Mac v247).

It’s important as we’ve made some adjustments to the settings in key areas and they may affect older projects:

  • Convert to HTML
  • GitHub Pages
  • FTP
  • Amazon S3 (new settings)

Some of these are fairly big changes, so if you use any of these features on old Mixture projects please make sure you read the details below and check out the relevant sections within the docs.

Convert to HTML

We’ve adjusted the settings and moved them into their own setting within the mixture settings file.

For more information please check out the updated docs for convert to HTML.

GitHub Pages

We’ve removed the GitHub pages settings completely from the mixture settings file and they have been pushed into the “.githubsettings” file.

This is beneficial for projects with multiple users as it allows each user to have and maintain their own settings without affecting the core project settings - you can also ignore this file from any version control you may use.

For more information please check out the updated docs for GitHub pages.

FTP

We’ve removed the FTP settings completely from the mixture settings file and they have been pushed into a new file “.ftpsettings”.

This is beneficial for projects with multiple users as it allows each user to have and maintain their own settings without affecting the core project settings - you can also ignore this file from any version control you may use.

For more information please check out the updated docs for FTP.

Amazon S3 (new!)

We’ve added support for Amazon S3 static hosting, simply create your bucket within your own Amazon account add your credentials and bucket name to “.s3settings” file within the root of your Mixture project and you can begin publishing to Amazon S3 from within Mixture tools.

For more information please check out the Mixture + Amazon S3 docs.

These releases have various stability improvements and bug fixes, including an update to libsass taking it to version 2.

Please checkout the changelogs for more information on each platform.
